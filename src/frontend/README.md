# Frontend

Client-side UI for the Gluten-Free Bakery app.

## Contents

- **`routes/`** — TanStack Router pages (URLs). Configured in `vite.config.ts` (`routesDirectory: frontend/routes`).
- **`components/`** — Layout (Header, Footer, ProductCard) and shadcn UI.
- **`lib/`** — Auth, cart, i18n, utilities.
- **`hooks/`** — Shared hooks (e.g. `use-mobile`).
- **`router.tsx`** — `createRouter` and error boundary.
- **`routeTree.gen.ts`** — Generated by TanStack Router (do not edit).
- **`styles.css`** — Tailwind entry.
- **`assets/`** — Images referenced by routes and `lib/images.ts`.

Data from the browser goes through **`@/backend/db/client`** (Supabase + RLS). See `src/backend/README.md` for server-only code.
